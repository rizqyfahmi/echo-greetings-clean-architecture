This is a [gin-gonic](https://gin-gonic.com/) project

## Prerequisite
1. go 1.20
2. gin 1.9.1

## Getting Started
1. Ensure that all modules are satisfied
   ```sh
   go mod tidy
   ```
2. Download all of the modules in the dependency graph
   ```sh
   go mod download
   ```
3. Run the project
   ```sh
   go run .
   ```
